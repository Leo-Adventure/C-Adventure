# Lambda 表达式
## 优点
1. 就地匿名定义目标函数以及对象，不需要额外写一个命名函数或者函数对象，更加直接写程序
2. 避免代码膨胀以及功能分散
3. 实现功能闭包

## 概念以及基本用法
语法形式
$$
[capture] (params) opt -> ret {body;};
$$

### 捕获列表
- [] 不捕获任何变量
- [&] 按照引用捕获外部作用域所有变量
- [=] 按照副本捕获外部作用域所有变量
- [=, &foo] 按值捕获外部作用域所有变量，但是按照引用捕获foo 变量
- [bar] 按值捕获 bar 变量，不捕获其他变量
- [this] 捕获当前类中的this 指针

注意按值捕获变量是无法进行对变量修改的，如果希望修改，则需要通过引用进行捕获。
因为按照 C++ 11 标准， lambda 表达式的 operator() 默认是 const 的。

没有捕获变量的 lambda 表达式可以直接转换为函数指针，反之则不能


注意 lambda 表达式是延迟调用的，也就是说
```cpp
    int a = 0;
    auto lam = [=]() -> int{return a;};
    a ++;
    cout << lam() << endl; // 输出 0

    int b = 0;
    auto lam_b = [&]() -> int(return b;);
    b ++;
    cout << lam_b() << endl; // 输出 1

```
